# 🤖 RaviBot

AI Chatbot with:
- Voice input 🎤
- Image analysis 📷
- News + Weather 🌤️
- Chat history 💬

## Features
- Guest chat (no login required)
- Login for saving chats
- AI powered responses

## Tech Stack
- Node.js
- Express
- MongoDB
- EJS
